这是针对BGI脚本的工具。它支持解包*.arc封包以及修改文本。
参考了https://github.com/xmoeproject/BGITool的ARC解包和https://github.com/marcussacana/EthornellEditor的文本修改工具
你也可以把这个当做一些工具的集成。
对一些代码做了修改，使其支持大部分BGIV2版本游戏。
有空再做GUI，那时可能使用起来会方便一点。

###注意：
	1、使用方法我直接在工具里写了。
	2、请将工具用于汉化。
